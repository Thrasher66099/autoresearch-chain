# simulations/

Adversarial and economic simulation scenarios for AutoResearch Chain.

## Status

Not yet populated. This directory will contain simulation definitions and test cases for:

- Multi-actor protocol behavior (competing proposers, validators, challengers)
- Adversarial scenarios (branch spam, challenge abuse, bad genesis proposals)
- Economic stress tests (reward starvation, domain creation pressure, fork proliferation)
- Successor-track migration scenarios

These simulations correspond to Phase 4 in the [Implementation Plan](../docs/implementation-plan.md).
